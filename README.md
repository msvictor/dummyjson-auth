## :octocat:Clone this repo

1. On command line terminal:

```sh
  git clone https://github.com/msvictor/dummyjson-auth.git
```

1. Go to folder:

```sh
  cd dummyjson-auth
```

3. Add the dependencies

```sh
  yarn
```

4. Run the app

```sh
  # on android device
  yarn android

  # on ios device
  yarn ios
```
